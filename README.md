# Online Image Editor
demo : (https://youtu.be/WBKbaMPGmeo)
A simple and powerful tool for editing bitmap or vectorial files, with the ability to estimate embroidery stitches amount.

## 🎨 Features

- **Bitmap Editing**: Edit raster images with ease
- **Vector Graphics**: Create and modify vectorial designs
- **Embroidery Estimation**: Calculate and estimate the amount of embroidery stitches needed for your designs
- **User-Friendly Interface**: Intuitive web-based editor
- **Real-time Preview**: See your changes instantly

## 🛠️ Tech Stack

- **Frontend**: JavaScript (63.6%), HTML (14.7%), CSS (19.7%)
- **Backend**: Node.js, Python (2%)
- **Server**: Express.js

## 📋 Requirements

- Node.js
- npm (Node Package Manager)
- Python 3.x (for embroidery stitches estimation)
